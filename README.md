# showdown-markdown-editor

A markdown editor using codemirror and previewer using showdown for Vue.js

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
